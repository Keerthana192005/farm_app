# Farm Fresh Vegetables App - Deployment Package

## Quick Start

### Windows:
```bash
start.bat
```

### Linux/Mac:
```bash
chmod +x start.sh
./start.sh
```

### Manual Start:
```bash
pip install -r requirements.txt
python run.py
```

## Access
After starting, open your browser and go to:
- http://localhost:5000

## Features
- Browse fresh vegetables
- Shopping cart functionality  
- Checkout system
